Example of console.warn

See the console for an example of console.warn. You can expose the console with a keyboard shortcut, as follows:

Mac

The keyboard command to invoke the console, which is a component of the developer tools, for Chrome, Firefox, Opera, and Safari is Command+Option+I.

Windows 10

For Chrome, Firefox, and Opera in Windows 10, the shortcut is Command+Option+I.

Fedora Linux

And for Chrome and Firefox in Fedora Linux, the shortcut is Command+Option+I.
